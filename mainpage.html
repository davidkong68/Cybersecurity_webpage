Type of cybersecurity threat



Phishing attack

What is phishing attack? Phishing attack is a type of social engineering attack that trick the victim into clicking a malicious link, and bring them into a replica of an official website. Once the victim enter the login credential, all the information will send to the hacker. Later, the hacker will use the retrieve login credential to login into the official website to transfer funds to the hacker or unauthorized purchase throught online website.



Denial of service attack

A Denail of Service (DoS) attack is an attack where the hacker uses a network of hijacked computers to flood the target server with tone of traffic which cause the server to crash, this cause the intended user unable to visit the website.

\

Ransomware attack

Ransomware is a type of malicious software (malware) that threatens to publish or blocks access to data or a computer system, usually by encrypting it, until the victim pays a ransom fee to the attacker. In many cases, the ransom demand comes with a deadline.



SQL attack

SQL (Structured Query Language) injection attack happen when hacker identifies vulnerable in the sql driven website, then the hacker performance the malicious SQL code into the database to grand access to itslef , once the hacker is inside the database it can basically alter the data or view sensitive information that store inside the database
